# Finance Pack

Finance Pack demonstrates how OpenPortalKit can adapt to a regulated, data-heavy domain without changing the industry-neutral core.

Allowed here:

- finance content type definitions
- finance dataset schemas
- finance validation rules
- finance dashboard cards
- finance templates
- finance sample data
- finance import examples

Not allowed:

- making core projects depend on this pack
- moving pack-specific entities into `OpenPortalKit.Kernel`
- requiring this pack for a generic portal to run
